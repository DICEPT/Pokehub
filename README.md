## 시작하기

### clone project
```bash
$ git clone git@github.com:side-project-pokehub/my-pokemon.git
```
### open project
```bash
$ cd my-pokemon
```
### install
```bash
$ yarn install
```
### start
```bash
$ yarn dev
```
